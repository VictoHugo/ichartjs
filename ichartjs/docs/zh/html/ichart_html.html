参数类型描述定义者
borderObject此组件中边框的属性样式。
此属性是一个对象，可用的属性如下：
@Option enable {boolean} 是否边框可用。(默认为false)
@Option color {String} 边框的颜色。(默认为'#BCBCBC')
@Option width {Number/Array} 边框的宽度,单位px。如果为数组，应该有4个值(上-右-下-左)指定其4个人边的宽度值。(默认为1)
@Option radius {Number} 边框的圆角值.(默认为5)
