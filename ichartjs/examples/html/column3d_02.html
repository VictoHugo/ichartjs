说明
这是一个3D柱形图例子，该例子展示了目前服务客户的情况。此例子中，配置了个性化的值轴。对于个性化值轴，关键在于能够知道全部数据中的最大值。设置其起始值(start_scale),最大值(end_scale),值间距(scale_space)。 当然，如果你的柱形图的值是不确定的，则组件会自动帮你计算最大值。
备注： 数据均为模拟，无实际参考价值。
var data = [ {name : '已服务客户数量',value : 30,color:'#bc6666'}, {name : '其中已服务VIP数量',value : 4,color:'#6f83a5'}, {name : '等待中数量',value : 20,color:'#76a871'}, {name : '处理中数量',value : 10,color:'#2ba5a4'} ]; $(function(){ new iChart.Column3D({ render : 'canvasDiv', data: data, title : 'Status Of Customer Service', width : 800, height : 400, coordinate:{ scale:[{ position:'left', start_scale:0, end_scale:40, scale_space:4 }] } }).draw(); });
